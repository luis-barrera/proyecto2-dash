## Demo local

Esta aplicación requiere de Python 3.

Para poder ejecutar la aplicación es necesario instalar las dependencias:

```shell
pip install -r requirements.txt
```

Luego, iniciamos la aplicación con:

```shell
pytohn app.py
```
